Place your 6 MP4 files in this folder using these exact filenames
(or update the `file` values in PROJECTS inside script.js to match your own names):

  reel-01.mp4   -> used as the hero background video + "Realtime Recommender Engine"
  reel-02.mp4   -> "Document Vision Pipeline"
  reel-03.mp4   -> "Support Copilot"
  reel-04.mp4   -> "Defect Detection at the Line"
  reel-05.mp4   -> "Forecasting Service"
  reel-06.mp4   -> "Voice Intent Router"

Tips:
- Keep each file reasonably compressed (H.264, under ~15MB) so the reel loads fast.
- The hero video (reel-01.mp4) plays muted + looping in the background, so a clip with
  clear visual motion (dashboards, terminals, model output) reads best there.
- You can freely edit titles, tags, and descriptions in script.js -> PROJECTS array.
